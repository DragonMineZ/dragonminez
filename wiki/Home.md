# DragonMineZ Developer Wiki

This wiki is for developers who want to help with DMZ modding.

If you are looking for player guides, use the main project docs instead on [Trello](https://trello.com/b/xewaqYee/dragonmine-z-trello).

## What you can do from this wiki

- Add or tune races, forms, sagas, wishes from config files, without building a separate addon.
- Add custom race models and textures, body types, eyes, hairstyles... Without building a separate addon.
